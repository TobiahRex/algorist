üìê The Four-Phase Learning Pipeline

1. Structure

What's the data layout? Where are pointers/indices? What state do we track? Draw the "before" picture.

2. Behavior

How do pointers move? What conditions trigger changes? When do we stop? Animate the transformation.

3. Verbose Code

Explicit class with state. Named helper methods. Clear variable names. Self-documenting structure.

4. Terse Code

Inline the helpers. Use idioms. Compress to interview-ready form. Keep mental model from Phase 1-2.
